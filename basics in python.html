Syntax in Python

Python Line Structure:

A Python program is divided into a number of logical lines and every logical line is terminated by the token NEWLINE. A logical line is created from one or more physical lines. A line contains only spaces, tabs, formfeeds possibly a comment, is known as a blank line, and Python interpreter ignores it. A physical line is a sequence of characters terminated by an end-of-line sequence (in windows it is called CR LF or return followed by a linefeed and in Unix, it is called LF or linefeed). Commenting

A comment begins with a hash character(#) which is not a part of the string literal and ends at the end of the physical line.
